# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Описание

"Новое русское вино" — это платформа для знакомства с эксклюзивной коллекцией вин. На сайте представлена информация о сортах, акциях и характеристиках напитков, отзывы клиентов и контакты для связи.

## Установка и запуск

1. Скачайте репозиторий:
   ```bash
   git clone https://github.com/Eugene571/new_russian_wine.git
   cd new_russian_wine

2. Убедитесь, что у вас установлен Python версии 3.8 или выше.

3. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

4. Запустите сайт:
   ```bash
   python main.py
   ```
5. Откройте сайт в браузере по адресу http://127.0.0.1:8000.
   
## Настройка путей к файлам

Для настройки путей к файлам данных и шаблонам можно использовать два способа:

### 1. Через командную строку

При запуске программы можно указать путь к файлам через аргументы:

```bash
python main.py --data path/to/your/wine.xlsx --template path/to/your/template.html
```
Если не указаны параметры, программа будет использовать значения по умолчанию:
- Путь к данным: wine.xlsx
- Путь к шаблону: template.html

### 2. Через переменные окружения

Для удобства можно настроить переменные окружения в файле `.env`. Пример содержимого `.env`:

```bash
WINE_DATA_PATH=path/to/your/wine.xlsx
TEMPLATE_PATH=path/to/your/template.html
```

| Белые вина   | Ркацители           | Ркацители     | 499  | rkaciteli.png         |                      |
| Красные вина | Черный лекарь      | Качич         | 399  | chernyi_lekar.png     |                      |
| Красные вина | Хванчкара           | Александраули | 550  | hvanchkara.png        |                      |
| Белые вина   | Кокур              | Кокур         | 450  | kokur.png             |                      |

| Напитки      | Коньяк классический |               | 350  | konyak_klassicheskyi.png |                      |

### Пример файла `wine.xlsx`

Сохраните данные в файл `wine.xlsx`. Пример файла находится в репозитории.

Если вы хотите использовать свои данные, создайте таблицу в аналогичном формате.

Изображения нужно разместить в папке `images`.

## Основные функции

- Просмотр коллекции вин, сгруппированных по категориям.
- Отображение информации о каждом вине: название, сорт, цена, наличие акций.
- Раздел отзывов от клиентов.
- Контактная информация и ссылки на социальные сети.
